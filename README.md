# 🌍 Racial Justice Education Resource Platform

## 📌 Overview
This is a multi-page web application designed to promote awareness and education about racial justice. The platform allows users to explore educational resources, take quizzes, and share personal experiences in an interactive and user-friendly environment.

---

## 🚀 Features
- 📖 Educational resources page for learning key concepts  
- 🧠 Interactive quiz system to test user knowledge  
- 🔐 Login page for user access  
- ✍️ Experience sharing page for user-generated content  
- 🔍 Browse experiences shared by others  
- 📞 Contact page for support and information  

---

## 🛠️ Tech Stack
- HTML  
- CSS (Tailwind CSS)  
- JavaScript  
- TypeScript (basic configuration with Vite)
